# Mods

Descarga:

- https://github.com/JcperezDev/minecraft-lab-mods/releases/latest

Instalacion:

1. Descarga `flacos-og-base-client.zip`.
2. Extraelo en cualquier carpeta.
3. Copia los `.jar` de `mods/` a `.minecraft/mods`.
4. Si quieres extras, descarga `flacos-og-extras-client.zip` y copia tambien esos `.jar`.
5. Usa NeoForge `1.21.11`.
